# 🕌 Majelis Taklim Miftahul Hidayah - Digital Hub

Selamat datang di repositori resmi **Majelis Taklim Miftahul Hidayah**. Proyek ini dirancang untuk menyatukan umat melalui teknologi, menyediakan informasi jadwal ibadah real-time, dan menjadi pusat informasi kegiatan majelis di Ciamis, Jawa Barat.

## 🚀 Fitur Utama

* **Jadwal Sholat Real-Time:** Sinkronisasi otomatis dengan API Aladhan (Metode Ikhbar NU).
* **Kalender Hijriah:** Penanggalan Islam yang disesuaikan dengan kondisi lokal Indonesia.
* **Moving Sky Background:** Efek visual bintang bergerak yang menenangkan dan dinamis.
* **Floating Action Navigation:** Akses cepat ke media sosial dan alat ibadah dalam satu klik.
* **Responsive Design:** Tampilan premium yang optimal di perangkat mobile maupun desktop.

## 🔗 Link Ekosistem

* **Website Jadwal Sholat:** [majelis-miftahul-hidayah.vercel.app](https://majelis-miftahul-hidayah.vercel.app/)

## 🛠️ Teknologi yang Digunakan

* **HTML5 & CSS3:** Struktur dan desain modern dengan teknik *Glassmorphism*.
* **JavaScript (Vanilla):** Logika waktu real-time, sistem partikel bintang, dan integrasi API.
* **Aladhan API:** Sumber data waktu sholat internasional.
* **Vercel:** Platform deployment untuk kecepatan akses maksimal.
